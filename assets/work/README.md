# Photos go here

Drop Osayi's photos into this folder named:

```
01.jpg  02.jpg  03.jpg  04.jpg  05.jpg  06.jpg  07.jpg  08.jpg
```

They'll appear automatically in the WORK grid on the site — no code changes needed.
Until a file exists, that slot shows an "ADD PHOTO" placeholder.

Tips:
- `.jpg`, `.png`, and `.webp` all work (keep the `01`, `02` … numbering).
- The grid mixes tall / wide / square tiles, so a mix of portrait and
  landscape shots looks best.
- Images are auto-converted to black & white, fading to full color on hover —
  that's intentional for the editorial look.
- Aim for high-res (long edge ~2000px) and compress before committing so the
  site stays fast.
